# IronPress

Projeto inicial com template basico de um website de uma usina metalúrgica.
Esse website tem como objetivo ser fundamental a empresa IronPress para divulgar o trabalho que é feito na empresa, e também o facil acesso a possiveis clientes a fazer orçamentos e ter o que desejam. Aqui fabricamos todo tipo de peças, automotivas, industriais, moldes, etc. Utilizamos de centro de usinagem, centro de torneamento e ferramentaria para fabricação.
